🪪 GitHub 令牌(PAT) 创建图文指南

全中文 · 照图点 · 菜鸟也能 3 分钟拿下来

⚠️ 你不用看懂英文界面，只要对着下面的图，点圈出来的位置就行。 拿到令牌后整串复制发给我，剩下的「建仓库 + 推送 + 上线 Render」我全包了。

0先确认你有 GitHub 账号

打开 github.com（已经被翻译成英文没关系，我们只看右上角）。

github.com
Sign up 用邮箱注册（一步步填，都是英文但都是名字/邮箱/密码） ② 有账号：点 Sign in 登录。登录后右上角会出现一个 圆形头像（如上图 U）
✅ 目标：登录后，右上角能看到你的圆形头像图标。能看到它，第 0 步就过了。

1点开右上角头像 → Settings（设置）

把鼠标移到右上角的圆形头像上，会弹出一个菜单，在最底下找到 Settings 点它。

github.com
Settings（图标是齿轮 ⚙），点它
💡 英文看不懂没关系，菜单里找带 ⚙ 齿轮 的那一行就是 Settings。

2左侧栏找到 Developer settings

进入 Settings 后，页面左边有一长串菜单。把滚动条拉到最底下，找到 Developer settings（开发者设置）点开。

github.com/settings/profile
Developer settings

3点 Personal access tokens → Tokens (classic)

点开 Developer settings 后，左边出现新菜单。点 Personal access tokens，再点下面的 Tokens (classic)。

github.com/settings/tokens
Tokens (classic) （经典令牌，最通用）

4点 Generate new token (classic)

右上角有个绿色按钮 Generate new token → 选 Generate new token (classic)。

github.com/settings/tokens
Generate new token (classic)

5填表：名字 + 勾选权限 + 有效期

新页面就填这几项，照着勾：

github.com/settings/tokens/new
✅ 只勾 2 个：reporead:user。别傻乎乎全勾，容易有安全风险。

6复制那串令牌，发给我

点完 Generate 后，页面会显示一长串以 ghp_ 开头的绿色字符。马上复制！这个页面关掉后就再也看不到了。

github.com/settings/tokens
复制 按钮，然后整串粘贴到我们的对话框发给我 ⚠️ 这串字符像密码，别截图发朋友圈；用完我会建议你去 Revoke 掉
📋 复制方法：选中那串字 → 按 Ctrl+C（Mac 用 ⌘+C），然后在聊天框按 Ctrl+V 粘贴。

R顺手拿一下 Render 的 API Key（同样给我）

打开 dashboard.render.com → 左下角点你头像 → Account Settings → 左边 API Keys → New API Key → 取个名字 → 生成后复制那串 rnd_ 开头的字符给我。

dashboard.render.com/u/settings
API Keys → New API Key → 命名 → 复制 rnd_ 开头那串 rnd_xxxxxxxxxxxxxxxxxxxx

📨 把下面两样一起发给我，我就开工：

  1. GitHub 令牌：ghp_xxxxxxxx...（整串）
  2. Render API Key：rnd_xxxxxxxx...（整串）

你还可以顺带说一句：仓库公开还是私有（默认公开）、仓库名（默认 feiyue-credit-system）。
收到后我会：① 用令牌在 GitHub 建仓库并推送代码 ② 用 Render Key 创建免费 Web 服务 ③ 把你的 *.onrender.com 上线地址发你。你就坐等网址就行 😎
