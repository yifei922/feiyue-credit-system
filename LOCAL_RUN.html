💻 本地运行图文指南（菜鸟版）

不绑卡、不上云，系统直接跑在你自己电脑上，数据也只存在你电脑里，最安全、最稳定。

✅ 你的电脑已经具备运行条件：Node 已装好、依赖已装好、启动脚本已备好。

1找到项目文件夹里的 start.bat

打开文件夹（就是存放这个系统的目录）：

C:\Users\40814\WorkBuddy\2026-07-19-17-04-34\

在里面找到这个文件，双击它：

📁 2026-07-19-17-04-34
├─ frontend/
├─ server/
├─ 🟢 start.bat ← 双击这个
├─ README.md
└─ ...

2看到黑窗口 + 浏览器自动打开

双击后会出现一个黑色命令行窗口（别关它），同时浏览器自动打开：

============================================
Feiyue Class 10 Credit System is starting
Open: http://localhost:3001
Login: admin / 123456
Close this window to stop the server.
============================================
看到这行就成功了：[server] 斐越十班学分系统已启动: http://localhost:3001

3登录使用

浏览器里输入账号：

  • 管理员：admin / 123456
  • 老师：teacher01 / 123456
  • 课代表：rep01(语文) · rep02(数学) / 123456
  • 学生：stu01~stu06 / 123456

登录后左侧菜单点 学生管理，就能导入/导出名单和成绩。

4怎么停止 / 怎么再开

停止：直接关掉那个黑色窗口即可。

再开：再双击 start.bat 就行。

⚠️ 黑窗口开着期间系统才在运行；关机或关窗口后别人（同局域网）就访问不了。想长期给大家用，就让这台电脑一直开着并运行着 start.bat。
📌 数据存在哪里？ 在你电脑的 server\data\credit.db 文件里，关掉再开数据都在。建议定期在「学生管理」里点导出名单 / 导出成绩做备份（复制一份到别处）。

🌐 想让同学在自己电脑也能用？

本地运行默认只有你这台电脑能访问。如果希望同班同学用各自设备访问，需要：

  • 方案 A：把项目文件发给同学，让他们也双击 start.bat（各用各的本地数据）；或
  • 方案 B：以后愿意绑卡了，再按 RENDER_GUIDE.html 部署到 Render，给大家一个网址。
