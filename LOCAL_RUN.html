💻 换电脑后本地运行图文指南（菜鸟版）

这个压缩包是便携版：已经把运行所需的 Node 和运行环境都打包进来了，新电脑不用装任何东西，解压后双击就能用。数据只存在你电脑里，最安全、最稳定。

✅ 压缩包里已包含：系统程序(server/)、已经做好的网页(frontend/dist/)、一键启动脚本(start.bat)、以及自带的 Node 运行时(runtime/node.exe)。

1把压缩包解压到一个新文件夹

例如：在桌面右键 → 新建文件夹，命名为 斐越系统，把 zip 解压进去。

⚠️ 一定要解压到空的新文件夹，不要覆盖你新电脑上已有的任何 WorkBuddy 文件，以免把记忆弄混。

解压后里面大概长这样：

📁 斐越系统/
├─ frontend/ ← 网页（已构建好）
├─ server/ ← 系统程序
├─ runtime/ ← 自带的 Node（不用管）
├─ 🟢 start.bat ← 双击这个
├─ README.md / LOCAL_RUN.html / RENDER_GUIDE.html
└─ ...

2双击 start.bat

找到 start.bat，双击它。会弹出黑色窗口（别关），浏览器自动打开：

============================================
Feiyue Class 10 Credit System is starting
Open: http://localhost:3001
Login: admin / 123456
Close this window to stop the server.
============================================
看到这行就成功了：[server] 斐越十班学分系统已启动: http://localhost:3001

3登录使用

浏览器里输入账号（密码都是 123456）：

  • 管理员：admin
  • 老师：teacher01
  • 课代表：rep01(语文) · rep02(数学)
  • 学生：stu01~stu06

登录后左侧菜单点 学生管理，就能导入/导出名单和成绩。

4怎么停止 / 怎么再开

停止：直接关掉那个黑色窗口即可。

再开：再双击 start.bat 就行。

⚠️ 黑窗口开着期间系统才在运行；想长期给大家用，就让这台电脑一直开着并运行着 start.bat。
📌 数据存在哪里？ 在你电脑的 server\data\credit.db 文件里。第一次双击 start.bat 会自动建好初始数据（6名学生/4门课程）。
💡 建议定期在「学生管理」里点导出名单 / 导出成绩做备份（复制一份到别处）。
🔒 关于记忆（重要）：这个压缩包不包含 WorkBuddy 的内部记忆文件，所以你新电脑上原有的记忆不会被覆盖，可以放心解压。

🌐 想让同学在自己电脑也能用？

  • 方案 A：把这个压缩包发给同学，让他们也解压后双击 start.bat（各用各的本地数据）；或
  • 方案 B：以后愿意绑卡了，再按 RENDER_GUIDE.html 部署到 Render，给大家一个网址。
